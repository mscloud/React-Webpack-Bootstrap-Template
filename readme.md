# React Webpack Bootstrap Template

## Introduction
This template serves as a starting point for a new react project

It contains:
* React JS
* Webpack
* Bootstrap V4 Alpha
* SASS
* Hot reloading

## Install
Install packages

``npm install``

## Development
To run local server

``npm run start``

View in browser: `localhost:8080`

## Production
To build production

``npm run build``

Everything is built in `/dist`
